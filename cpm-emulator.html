• Microsoft BASIC-80 (MBASIC) or compatible
• Support for PRINT USING, TAB(), and LPRINT
• Approximately 40KB of memory

Quick Start Guide

To run MCDRAG on real CP/M:

  • Start CP/M emulator
  • Type MBASIC
  • Load the program with LOAD "MCDRAG.BAS"
  • Type RUN

About CP/M

CP/M (Control Program/Monitor) was the dominant operating system for 8-bit computers in the late 1970s and early 1980s.

Historical Note: MCDRAG was written in 1974, predating even CP/M! It would have originally run on early minicomputers and timesharing systems.
