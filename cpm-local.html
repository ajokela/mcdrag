Quick Start Guide

  1. Click in the emulator window
  2. Press Enter if you see a prompt
  3. Type B: and press Enter
  4. Type DIR to see files
  5. Type MBASIC for Microsoft BASIC

CP/M Commands

Essential CP/M commands:

A:          - Switch to drive A
B:          - Switch to drive B  
DIR         - List files
TYPE file   - Display text file
MBASIC      - Start Microsoft BASIC
STAT        - Show disk status
ERA file    - Delete file

BASIC Commands

Once in MBASIC:

NEW         - Clear program
LIST        - Show program
RUN         - Execute program
SAVE "name" - Save program
LOAD "name" - Load program
SYSTEM      - Exit to CP/M
