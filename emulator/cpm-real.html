Quick Instructions

  1. Click in the emulator window to focus
  2. Type B: to switch to drive B
  3. Type MBASIC to start Microsoft BASIC
  4. In BASIC, type commands to enter MCDRAG
  5. Type RUN to execute
Note: This is a real Intel 8080 emulator running CP/M 2.2
