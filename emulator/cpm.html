CP/M MBASIC Instructions

This is a real CP/M emulator running Microsoft BASIC-80 (MBASIC) in your browser!

  • Click in the terminal to focus and start typing
  • Type MBASIC and press Enter to start Microsoft BASIC
  • Type LOAD "MCDRAG" to load the program
  • Type RUN to execute the program
  • Type LIST to view the program listing
  • Type SYSTEM to exit back to CP/M
  • Use Ctrl+C to break program execution

Note: This is running actual CP/M 2.2 with MBASIC 5.21 - the same environment from the 1970s-80s!
